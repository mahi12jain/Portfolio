Facial Recognition API for Employee Attendance

Developed a facial recognition API using base64 encoding for secure and efficient employee authentication.

This API enables seamless face matching by encoding images in base64 format and verifying them against stored employee records. It is integrated into the company’s attendance system to automate check-in and check-out processes, eliminating manual interventions.

By leveraging biometric-based authentication, the system enhances security, prevents unauthorized access, and ensures accurate attendance tracking.

Mail Classification using LLMA3

An AI-powered email classification system utilizing LLMs for efficient inbox management.

This project implements an email classification agent leveraging the Ollama model from the Langchain Community. It categorizes emails as 'important', 'casual', or 'spam' while also generating concise responses based on email priority, ensuring an organized and efficient email management system.

Key Features:

  • Developed using Python and specialized language models (LLMs).
  • Intelligent email classification for automated sorting.
  • AI-powered agent generates quick responses based on priority.
  • Demonstrates expertise in NLP, AI delegation, and sequential processing.
